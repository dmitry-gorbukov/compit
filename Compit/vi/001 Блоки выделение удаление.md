## Выделение блока текста в vi/vim:

**В визуальном режиме:**
- `v` - вход в визуальный режим (побайтовое выделение)
- `V` - выделение целых строк
- `Ctrl+V` - блочное выделение (прямоугольник)

**Примеры выделения:**
```bash
v + движения курсора  # выделить символы
V + j/k               # выделить строки
Ctrl+V + движения     # выделить блок
```

## Удаление блока текста:

**После выделения:**
- `d` - удалить выделенное
- `x` - удалить выделенные символы

**Без выделения:**
- `dd` - удалить текущую строку
- `5dd` - удалить 5 строк
- `d}` - удалить до конца параграфа
- `dG` - удалить до конца файла

## Быстрое перемещение курсора:

**Начало/конец файла:**
- `gg` - в начало файла
- `G` - в конец файла
- `:1` - в первую строку
- `:$` - в последнюю строку

**Начало/конец строки:**
- `0` - в начало строки
- `^` - на первый не-пробельный символ строки
- `$` - в конец строки

**Другие полезные перемещения:**
- `Ctrl+F` - страница вниз
- `Ctrl+B` - страница вверх
- `w` - следующее слово
- `b` - предыдущее слово
- `}` - следующий параграф
- `{` - предыдущий параграф

## Пример работы с блоками:
1. `Ctrl+V` - войти в блочный режим
2. Выделить нужный блок стрелками
3. `d` - удалить блок
4. `y` - скопировать блок
5. `p` - вставить после курсора

**Примечание:** Для выхода из любого режима нажми `Esc`
